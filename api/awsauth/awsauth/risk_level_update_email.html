Last Tuesday we modified our risk levels to include a 5th level - Severe Outbreak or Extreme risk. On Monday, December 8th we will be updating the API to reflect this.

If you have any code depending on 4 risk levels, you will need to update it to include the fifth risk level.

This change directly affects the fields riskLevels.overall and riskLevels.caseDensity.

For more details on the new risk levels and how to update, see the risk level update documentation.
